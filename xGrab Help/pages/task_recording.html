Creating a Recording


Creating a new recording is a trivial task. You may, however, want to customize the default compression options. Please follow the steps below to begin a recording session and choosing its compression settings.

To create a recording do one of the following:

  • Select "Begin Capture..." from the "Capture" menu
  • Select "Begin Capture..." from the "xG" menu

To optimize compression settings for performance:

  1. Open xGrab's Preferences
  2. Select "Apple Animation" from the codec list
  3. Select 12 frames per second
  4. Select key frame every 300 seconds
  5. Select medium quality
  6. Make sure the output resolution matches the screen resolution

If you still need to improve capture performance we suggest you decrease your screen resolution to a common capture value. Try one of the following:

  1. 800x600
  2. 1024x768
